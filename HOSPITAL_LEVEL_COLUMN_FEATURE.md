# 医院等级独立列功能说明

## 📋 功能描述

在医院管理页面的表格中，将医院等级从"医院信息"列中独立出来，作为一个单独的列显示，使等级信息更加突出和易于查看。

## ✅ 修改内容

### 1. 表格结构调整

#### 修改前
```
[医院信息] - 包含：医院名称、等级徽章、地址等
[PET-CT 设备]
[价格与状态]
[是否合作]
[操作]
```

#### 修改后
```
[医院信息] - 包含：医院名称、地址等（移除等级徽章）
[医院等级] ⭐ 新增独立列
[PET-CT 设备]
[价格与状态]
[是否合作]
[操作]
```

### 2. 代码变更

#### 位置调整
- **原位置**: "医院信息"列内包含等级徽章
- **新位置**: 独立的"医院等级"列，位于"医院信息"和"PET-CT 设备"之间

#### 列配置
```vue
<!-- 医院等级列 -->
<el-table-column 
  label="医院等级" 
  prop="level" 
  width="120" 
  align="center"
>
  <template #default="scope">
    <span class="hospital-level-badge" :class="getLevelClass(scope.row.level)">
      {{ scope.row.level || '未分级' }}
    </span>
  </template>
</el-table-column>
```

### 3. 视觉效果

#### 等级徽章样式保持不变
- 三级甲等：绿色渐变
- 三级乙等：蓝色渐变
- 二级甲等：橙色渐变
- 二级乙等：红色渐变
- 一级甲等/乙等：灰色渐变
- 综合医院：紫色渐变
- 影像中心：粉色渐变

## 🔍 使用说明

### 查看医院等级
1. 在医院列表中找到"医院等级"列
2. 查看彩色徽章显示的等级信息
3. 不同颜色代表不同的医院等级

### 等级含义
- **三级甲等**（绿色）：最高等级医院
- **三级乙等**（蓝色）：高等级医院
- **二级甲等**（橙色）：中等医院
- **二级乙等**（红色）：基础医院
- **一级甲等/乙等**（灰色）：基层医院
- **综合医院**（紫色）：综合性医疗机构
- **影像中心**（粉色）：专业影像机构
