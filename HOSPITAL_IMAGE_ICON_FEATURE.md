# 医院图片图标功能说明

## 📋 功能描述

在医院管理页面的医院列表中，为有封面图片的医院添加了一个图片图标。当鼠标悬停在图标上时，会显示医院的封面图片预览。

## ✅ 实现效果

### 1. 视觉效果
- 在医院等级徽章旁边显示一个小型的图片图标（📷）
- 图标采用渐变蓝色背景，与整体设计系统保持一致
- 鼠标悬停时图标会轻微放大，提供交互反馈

### 2. 交互体验
- 鼠标悬停在图标上时，显示医院封面图片的预览
- 预览图片最大尺寸为 400x300px，保持图片比例
- 预览框带有圆角和阴影效果，提升视觉层次

### 3. 响应式设计
- 图标自动适应不同的屏幕尺寸
- 支持 flex-wrap 换行，避免在小屏幕上布局错乱

## 🔧 技术实现

### 1. 使用的组件
- **ElTooltip**: Element Plus 的提示框组件，用于显示图片预览
- **SVG 图标**: 使用照片图标表示图片功能

### 2. 代码结构

```vue
<!-- 医院图片图标 -->
<el-tooltip 
  v-if="scope.row.cover" 
  placement="top"
  :offset="10"
  class="hospital-cover-tooltip"
>
  <template #content>
    <div class="hospital-cover-preview">
      <img :src="scope.row.cover" :alt="scope.row.title" />
    </div>
  </template>
  <div class="hospital-cover-icon">
    <svg>...</svg>
  </div>
</el-tooltip>
```

### 3. 样式特点

#### 图标样式
- 尺寸：20x20px
- 背景：渐变蓝色 (#F0F9FF → #E0F2FE)
- 悬停效果：背景变深、放大 1.1 倍
- SVG 图标：14x14px，青色描边

#### 预览样式
- 最大宽度：400px
- 最大高度：300px
- 圆角：8px
- 阴影：柔和的深度阴影
- 图片适配：object-fit: cover 保持比例

## 📊 显示逻辑

### 显示条件
- 只有当医院有 `cover` 字段时才显示图标
- 没有封面图片的医院不显示图标

### 位置布局
图标位于医院等级徽章之后，与医院名称在同一行显示：
```
[医院名称] [等级徽章] [图片图标]
```

## 🎨 设计规范

### 颜色搭配
- 图标背景：蓝色渐变系，与主题色一致
- 图标边框：青色 (#0891B2)，呼应整体设计
- 悬停状态：更深的蓝色渐变，提供视觉反馈

### 动画效果
- 过渡时间：0.3s
- 缓动函数：ease
- 悬停变换：scale(1.1) 轻微放大

## 💡 用户体验优化

### 1. 可访问性
- 图片包含 alt 属性，使用医院名称作为替代文本
- 图标大小适中，易于点击和交互

### 2. 性能考虑
- 仅在需要时加载图片（悬停时）
- 限制预览图片的最大尺寸，避免影响页面性能

### 3. 兼容性
- 使用标准的 CSS 属性，兼容现代浏览器
- Element Plus 组件保证跨浏览器一致性

## 📝 使用说明

### 查看医院图片
1. 在医院列表中找到有图片图标的医院
2. 将鼠标移动到图片图标上
3. 查看医院封面图片的预览

### 隐藏图片图标
如果医院没有设置封面图片，图标将不会显示。

## 🔍 示例场景
